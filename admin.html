Solo administradores del proyecto

Este panel permite revisar el estado de las encuestas de Proyecto PAUSA y descargar las respuestas en formatos listos para análisis estadístico.

Acceso permitido únicamente para cuentas administradoras de Proyecto PAUSA autenticadas con Google.

Cualquier otra cuenta de Google será bloqueada para la lectura de datos.

Encuestas en Firebase

Para cada encuesta se muestran el número de respuestas guardadas en Firestore y accesos directos para descargar:

  • CSV datos: solo variables de respuesta, listo para importar a R / Python / SPSS.
  • CSV datos + paradata: incluye tiempos por ítem, orden de presentación y metadatos de navegador.
